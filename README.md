# PDF to Image Converter

A lightweight, browser-based tool to instantly convert PDF pages into high-quality JPG or PNG images without any server-side processing.

## Features
*   **100% Client-Side**: All conversions happen in your browser. Your files are never uploaded to a server, ensuring total privacy.
*   **Multi-Format Support**: Export pages as either `.jpg` or `.png` files.
*   **Batch Processing**: Automatically processes every page of your PDF into individual image files.
*   **Zero Dependencies**: Stand-alone HTML/JS tool that requires no installation or complex setup.
*   **High Quality**: Adjustable scaling to ensure sharp text and clear graphics.

## Built With
*   **HTML5/CSS3**: For the responsive user interface.
*   **JavaScript**: For core conversion logic.
*   **[PDF.js](https://mozilla.github.io)**: Mozilla's open-source library for parsing and rendering PDFs.

## How to Use
1.  **Clone** or download this repository to your local machine.
2.  Open `PDF2JPG.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
3.  **Upload** your PDF by clicking the file selector or using drag-and-drop.
4.  **Select** your desired output format (JPG or PNG) and quality settings.
5.  Click **Download** and download your images individually or as a single ZIP file.
